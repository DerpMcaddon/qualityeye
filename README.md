# QualityEye Demo 👀

Demo website sederhana untuk simulasi AI Quality Control berbasis Computer Vision.

Project ini dibuat sebagai prototype presentasi untuk menunjukkan bagaimana AI dapat membantu proses inspeksi produk di industri manufaktur, garmen, UMKM produksi, dan packaging.

---

# Fitur Utama

* Upload gambar produk
* Preview gambar sementara
* Simulasi inspeksi AI
* Random hasil QC:

  * Lulus QC
  * Cacat terdeteksi
  * Perlu cek manual
  * Error simulasi
* Log inspeksi otomatis
* Statistik inspeksi sederhana
* UI industrial dark mode
* Responsive untuk desktop dan mobile

---

# Tech Stack

* HTML
* CSS
* Vanilla JavaScript

# Cara Demo

1. Upload foto produk
2. Klik tombol "Mulai Inspeksi AI"
3. Sistem akan mensimulasikan proses AI inspection
4. Hasil random akan muncul
5. Data otomatis masuk ke log QC


# Struktur Sistem

Frontend only.

Tidak ada:

* Database
* Backend server
* API AI asli
* Machine learning model

Seluruh output dibuat menggunakan random generator JavaScript agar ringan dan mudah dipresentasikan.
